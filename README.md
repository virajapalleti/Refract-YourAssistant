# Refract

A PDF assistant that explains document concepts adapted to the user's personal writing style.

## Quickstart

1. Clone the repo
2. Run the setup script for your platform:
   - Windows: `setup.bat`
   - Mac/Linux: `setup.sh`
3. Activate the venv:
   - Windows: `venv\Scripts\activate`
   - Mac/Linux: `source venv/bin/activate`
4. Run `python validate_profiles.py` to check that everything is working
